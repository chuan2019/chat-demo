Welcome To the Customer Service Chat System

As an Analyst, you have to be registered. To register, you need to provide your nickname and password to the system administrator. He/She will add your nickname and password into the system, such that you can login to receive chatting request(s) from clients.

As a client, you can request to start chatting with an analyst. What you need is just a nickname whent send the request. If an analyst is available, a session will be created for you, and you can start chatting with the connected analyst. Otherwise, you will get an error message.

Due to time constraint, current version of the app does not have UI support yet, except for the analyst registration functionality. All chatting services are provided via APIs.
